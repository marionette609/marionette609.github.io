Measurement Info

Software

  • REW (Beta version)
  • 48khz Sampling Rate for Input/Output
  • Script used for REW API

Equipment Used

  • E1DA Cosmos ADCiso
  • E1DA Cosmos Scaler
  • If amplifier measurement, Schiit Modius E was used as signal generator

FFT/Distortion Settings

  • Blackman-Harris 7 Window
  • 32k Length
  • 4 Averages, No Smoothing
  • 20hz LP, 21khz HP
  • Use AES17-2015 Standard Notch
Viewer Options

The following are available as url parameters

  • zMin: Set the minimum displayed range for Z axis in 3D viewers (Default -130)
  • zMax: Set the maximum displayed range for Z axis in 3D viewers (Default -40)
  • targetPlot: Changes data shown in surface viewer and heatmaps. Can be thd, thdPlusN, or multi
  • smoothHeat: Enable smoothing for heatmaps, can be true or false
  • xLabel: Changes unit label for X axis
